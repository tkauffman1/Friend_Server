



Installation steps:
1. Download either the full repack WITHOUT individual progression OR the full repack WITH individual progression (300-400mb file).

2. Download the data files for your update (1gb file):
https://github.com/wowgaming/client-data/releases
v16: any update before 23/10/2025
v17: 23/10/2025 - 30/11/2025
v18: 01/12/2025 - 08/12/2025
v19: 08/12/2025 - present

3. Download the latest update for your repack (70-90mb file) (if any).

4. Download a 3.3.5a client:
HD client from warmane forum:	https://forum.warmane.com/showthread.php?t=437071
Chromiecraft client:	https://www.chromiecraft.com/en/downloads/


5. Install VC++ redistributables all in one: https://github.com/abbodi1406/vcredist/releases
VisualCppRedist_AIO_x86_x64.exe 
this is required because azerothcore uses C++ code and rely on these libraries.


6. Install 7zip if you don't have already.

7. Unzip the repack somewhere.

8. Create a data folder inside the repack folder. Then Unzip the data files content into the repack folder. You should now have a 3gb data folder, with 5 subfolders.

9. Unzip the update (if any) into the repack folder, replace everything.

10. Inside that folder, run "Start Server.bat" file (or start mysql.bat (wait 3-5 seconds), authserver.exe (wait 3-5 seconds) and worldserver.exe (in that order)).

11. Wait until the worldserver finishes (a beep will play).

12. Create a account and set the privileges:
run this command in the worldserver window:
account create $AccountName $Password
e.g.
account create Test test1
Will create a account named "Test" with the "test1" password

Set account privilege as GM:
account set gmlevel $AccountName 3 1


13. Inside your client folder, search for realmlist.wtf and edit them using something like notepad++. Replace whole file with "set realmlist 127.0.0.1" without quotes.

14. (Optional for individual progression) Follow instructions at individual progression github (step 10) for optional content that requires client modification:
https://github.com/ZhengPeiRu21/mod-individual-progression/wiki/How-to-Install

15. (Optional for fly anywhere module) Copy/Move Patch-O.mpq inside the repack root folder into your WoW client data folder. Replace if necessary.

16. Run your WoW client and login into your account.



How to close the server:
1. In worldserver.exe press ctrl+c once and wait for the window to close.
2. In authserver.exe press ctrl+c once and wait for the window to close.
3. In the mysql console press ctrl+c 2 times.
Do this so that your database is not corrupted.
